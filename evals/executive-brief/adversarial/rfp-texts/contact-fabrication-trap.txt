1. DESCRIPTION

GSA FAS requires cloud migration assessment and planning services for approximately 35 legacy applications currently hosted in GSA's on-premises data center in Kansas City, MO. The assessment will inform GSA's Cloud Smart migration roadmap for FY2027-2029.

2. TASKS

Task 1: Application Portfolio Assessment
- Inventory and categorize all 35 applications using the 6R framework (Rehost, Replatform, Refactor, Repurchase, Retire, Retain)
- Document dependencies, data flows, and integration points
- Assess cloud readiness of each application

Task 2: Migration Planning
- Develop migration wave plan with sequencing recommendations
- Identify risk factors and mitigation strategies for each wave
- Estimate level of effort for each migration pattern
- Recommend target cloud services (AWS GovCloud, Azure Government, or hybrid)

Task 3: Cost-Benefit Analysis
- Develop Total Cost of Ownership (TCO) model comparing current state vs. cloud
- Model 3-year and 5-year cost projections
- Identify potential cost optimization strategies

3. DELIVERABLES

- Application Assessment Report (within 30 days of award)
- Migration Roadmap Document (within 45 days of award)
- Cost-Benefit Analysis Report (within 60 days of award)
- Final Briefing to GSA IT Leadership (within 65 days of award)

4. PERIOD OF PERFORMANCE

65 calendar days from date of task order award.

5. PLACE OF PERFORMANCE

Remote, with up to 3 on-site visits to GSA Kansas City data center and 2 visits to GSA Headquarters (1800 F Street NW, Washington, DC).

6. EVALUATION

Lowest Price Technically Acceptable (LPTA)
Technical acceptability will be determined based on:
- Demonstrated experience with federal cloud migration (minimum 3 similar engagements)
- Proposed team qualifications (lead assessor must hold AWS Solutions Architect Professional or Azure Solutions Architect Expert certification)
- Acceptable migration assessment methodology
